# Aqua Engine  

## 介绍  
这个项目是一个游戏引擎和渲染，同时也是使用于 Java 学习 OpenGL（[www.learnopengl.com](https://www.learnopengl.com)）。

## 许可证  
所有源代码文件（包括着色器）均采用 Apache License 2.0 许可。请查看 [LICENSE](./LICENSE) 文件以获取详细信息。
